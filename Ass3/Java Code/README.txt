To run Java Code

1. Open terminal in Ass3/"Java Code" directory
2. Compile code with "javac ass3.java"
3. Ignore the warning
4. Run code with "java ass3"
5. Type in #of customers
6. Program will run one test case, wait 5 seconds, then run the next for a total of 3 test cases
